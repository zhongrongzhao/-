# 鱼蛋文件分割合并工具
### 编程语言：
  C#
### 框架：
  .Net Framwork 4.7.2
### 简介：
  让一个文本文件按自定大小分割为多个文本文件，<br>
  又可以让多个文本文件合并为一个文本文件<br>
